# WF_DSS_SCIENTIFIC_INVESTIGATION
**D.S.S — Department of Sovereign Science**
**Workflow: From Question to Sovereign Scientific Position**

---

## WHAT THIS WORKFLOW IS

The process for investigating any scientific question through the sovereign lens — applying rigorous methodology while remaining open to frameworks outside mainstream institutional science. Every investigation produces a documented position with honest confidence levels.

---

## TRIGGER

- A scientific question arising from any department
- A mainstream scientific claim that needs evaluation
- An alternative or suppressed scientific framework that needs investigation
- An ancient knowledge system that needs study on its own terms
- A question about consciousness, cosmology, biology, or physics

---

## THE WORKFLOW — STEP BY STEP

### STAGE 1 — QUESTION FORMULATION (Nigredo)
**Define exactly what is being asked.**

```
ACTION:
State the scientific question precisely.
- What specifically are we trying to understand?
- What is the mainstream position on this?
- What alternative positions exist?
- What would it mean to answer this question?

SKILLS TO LOAD: skill_source-evaluation.md
OUTPUT: dss_question_[topic]_formulation.md
```

---

### STAGE 2 — FRAMEWORK MAPPING (Albedo)
**Identify which frameworks address this question.**

```
ACTION:
Map the relevant frameworks:

1. MAINSTREAM SCIENCE: What does institutional science say?
   What is the evidence? What are the limitations?
   What anomalies does the mainstream framework not explain?

2. ALTERNATIVE FRAMEWORKS: What do non-mainstream models say?
   (Plasma cosmology, electric universe, expanding earth,
   pre-colonial knowledge systems, etc.)
   What is their evidence? What are their limitations?

3. ANCIENT KNOWLEDGE: What did pre-colonial civilizations
   understand about this? On what evidence?
   (Kemetic, Babylonian, Mesoamerican, Indian, Chinese)

4. EVIDENCE GAPS: Where is no framework adequate?
   What remains genuinely unknown?

OUTPUT: dss_frameworks_[topic]_map.md
```

---

### STAGE 3 — EVIDENCE EVALUATION (Citrinitas)
**Apply source tier standards across all frameworks.**

```
ACTION:
Route through D.R.D evidence extraction standards:
- Tier each source regardless of which framework it supports
- Log contradictions between frameworks at full strength
- Identify where genuine consensus exists across frameworks
- Identify where genuine disagreement exists and why

SKILLS TO LOAD: skill_source-evaluation.md, skill_framework-synthesis.md
OUTPUT: dss_evidence_[topic]_evaluation.md
```

---

### STAGE 4 — SOVEREIGN POSITION (Rubedo)
**State what is actually known, probable, and uncertain.**

```
ACTION:
Build the sovereign scientific position:
- ESTABLISHED: Multiple high-tier sources across frameworks agree
- PROBABLE: Strong evidence but not definitive
- POSSIBLE: Evidence exists but is thin or contested
- OPEN: Genuinely unknown — no framework has a strong answer
- REJECTED: Claims that do not survive evidence evaluation

The sovereign position is honest about uncertainty.
It does not replace mainstream dogma with alternative dogma.

OUTPUT: dss_position_[topic]_v1.md
```

---

### STAGE 5 — DOCUMENTATION AND CROSS-LINK
**Make the investigation retrievable and useful.**

```
ACTION:
- File all outputs in D.S.S
- Cross-link to requesting department
- Cross-link to D.R.D if deep research was conducted
- Cross-link to D.B.S if the topic relates to body/health science
- Update D.S.S context file with new active investigation

OUTPUT: Complete investigation file trail
```
