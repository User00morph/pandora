# D.C.E Stage 2 — ALBEDO: Expansion and Examination

## Purpose
Take the raw idea and determine: what is the core truth it points to, what distortions could enter, and what form fits the content. The idea earns its structure here.

## Inputs
- Filed idea from `D.C.E/ideas/` (Stage 1 output)
- D.R.D reconstruction if this is research-backed content (load the specific file)
- Load: `skill_concept-distillation.md`

## Process
1. **Identify the core truth.** Strip the idea to its irreducible claim. One sentence. If you cannot reduce it to one sentence, the core is not yet clear.

2. **Map the distortions.** What could make this piece misleading, unclear, or misaligned with its actual intention?
   - What is the audience most likely to misunderstand?
   - What is the most common wrong version of this idea?
   - What evidence gap could weaken the claim?

3. **Choose the format.** Load `format-patterns.md`. Match idea depth and complexity to format capacity.

4. **Determine depth.** What does this piece need to actually land? Surface (one clear point)? Medium (evidence + framework)? Deep (full historical-to-modern arc)?

5. **Write the concept document:**
```
CORE TRUTH: [one sentence]
DISTORTIONS TO AVOID: [bullet list]
FORMAT: [specific format]
SERIES: [which series]
DEPTH: surface / medium / deep
OUTLINE SEED: [3-5 bullets: opening beat → midpoint → close]
RESEARCH NEEDED: [yes/no — if yes, file D.R.D request before Stage 3]
```

## Done looks like
A filed concept document with all fields complete. Research sourced or confirmed available. Ready to script.

## Output goes to
`D.C.E/concepts/output/dce_concept_[title]_v1.md` → Stage 3 (Citrinitas)
