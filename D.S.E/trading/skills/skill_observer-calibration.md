## PRE-SESSION CALIBRATION (Run before opening any chart)

### PHASE 1 — BODY CHECK (2 minutes)

Sit still. Breathe. Answer these questions honestly:

```
□ Am I physically rested? (Less than 6 hours sleep = reduced cognitive function = no trading)
□ Am I emotionally neutral? (Any strong emotion — anger, anxiety, excitement — flags this)
□ Am I financially anxious about this session? (If yes, the scarcity state is active — do not trade)
□ Do I have competing mental demands today? (Urgent life situations = divided attention = no trading)
```

**If any answer is YES:** Do not trade today. Log: "Observer not present. Session skipped." This is a valid session outcome. The system operates on many sessions. One session skipped is not a loss — it is risk management at the most fundamental level.

**If all clear:** Proceed.

---

### PHASE 2 — PRIOR SESSION RELEASE (2 minutes)

If there were trades in the previous session:

```
□ Have I completed the post-trade log for those trades?
□ Am I carrying any emotional residue from those trades?
    (Anger at a loss? Excitement from a win? Urgency to "do better" today?)
□ Is today's session completely separate in my mind from yesterday's results?
```

**If residue detected:** Write one sentence describing the residue. Then write: "This is data from the previous experiment. The current session is a new experiment. The new experiment begins from zero." Read it aloud if needed. Release it explicitly.

**If clean:** Proceed.

---

### PHASE 3 — THE FIVE-LAYER FIELD READ (8 minutes)

This is the intellectual component. Read quickly but thoroughly.

```
LAYER 5 — OBSERVER CHECK (30 seconds)
  Am I on the Aries → Pisces wheel?
  (Conditions specified before price? Hypothesis formed from the field? No need to trade?)
  → YES = Observer position confirmed. Proceed.
  → NO = What is pulling me? Identify it. Release it. Re-check.

LAYER 4 — ASTRONOMICAL / CYCLIC CHECK (2 minutes)
  What is the Moon phase today?
  What planetary events are occurring today or within 3 days?
  What cycle phase is the market in? (Crisis / Polarization / Sweep)
  What is the dominant Ray expressing?
  → Log the answers in one line each.

LAYER 3 — EXTERNAL REALITY CHECK (2 minutes)
  What economic events are scheduled today?
    → Check: forexfactory.com or investing.com economic calendar
    → High-impact events: red-flag days (NFP, FOMC, CPI) = maximum caution
  What is DXY doing on the daily chart? (1 glance — up/down/ranging)
  Any overnight news that changes the macro narrative?
  → Log in one line.

LAYER 2 — COLLECTIVE CONSCIOUSNESS CHECK (2 minutes)
  What is the daily bias for each pair being watched?
    → Bullish / Bearish / No bias (from the structure)
  Where is the crowd positioned? (retail sentiment from broker dashboard)
  What emotional state does the chart feel like right now? (Sweep energy? Crisis energy?)
  → Log in one line per pair.

LAYER 1 — STRUCTURAL CHECK (2 minutes)
  For each pair being watched:
    → What is the daily structure? (HH+HL / LL+LH / ranging)
    → Where are the key levels? (liquidity pools, OBs, FVGs)
    → Where is the discount zone? (below 50% of the last swing for longs)
  → Draw these on the chart before any analysis begins.
```

### PHASE 4 — THE SESSION TRIAD (1 minute)

From STIS PRD v3 §12. Three questions applied to today's market:

```
SUN of today's session (the present problem):
  "What is the dominant question the market is trying to solve today?"
  Example: "Is the Fed going to pause or cut? Is risk appetite returning or fading?"
  Write one sentence.

RISING of today's session (the actual field):
  "What is actually being brought forward beneath the headline narrative?"
  Example: "The 30-year yield at 18-year highs is the real field — dollar credibility."
  Write one sentence.

MOON of today's session (the dead narrative):
  "What story is everyone still trading that has already expired?"
  Example: "The 'soft landing is assured' narrative — the bond market has already contradicted it."
  Write one sentence.
```

### PHASE 5 — THE HYPOTHESIS (before any chart is opened for analysis)

For each pair being considered for trading today:

```
CAN I FORM A VALID HYPOTHESIS?

A valid hypothesis requires:
  □ A named Ray expression ("This is an R[_] move")
  □ A named cycle phase ("during the [Sweep/Polarization/Crisis] phase")
  □ Layer 3 confirmation ("supported by [specific data — COT, yield spread, DXY position]")
  □ A specific entry condition ("confirmed when price does [X] at level [Y]")
  □ A defined stop ("invalidated if price closes [below/above] [level]")
  □ A defined target ("target is [level] for [R:R ratio] risk-reward")

If all six components can be stated → hypothesis is valid → watch for the condition.
If any component is missing → no hypothesis → no trade → observation only.
```

## POST-TRADE CALIBRATION (Run after every material trade — win or loss)

A "material trade" is any trade that used 0.5% or more of account equity as risk.

### STEP 1 — IMMEDIATE (within 5 minutes of close)

```
□ Is there emotional residue from this trade?
    (Relief, anger, excitement, despair)
    → Name it explicitly: "I feel [___] because [___]."
    → This is not a problem — it is information. Name it and release it.

□ Was the stop honored? (If yes: the system worked, regardless of outcome)
□ Was the position sized correctly? (1-2% of equity max)
□ Was the entry trigger valid? (A specific candle signal at a specific level)
```

### STEP 2 — THE HYPOTHESIS ASSESSMENT (within 30 minutes of close)

```
□ What was the hypothesis? (Restate it from the Trade Decision Log)
□ Was the Ray read correct?
    → If yes: what confirmed it? Add to the Ray-expression library.
    → If no: what misidentified it? Note for the pattern library.
□ Was the cycle phase correct?
    → If yes: the timing framework is working.
    → If no: what phase was it actually in? Why was it misread?
□ Was the entry trigger valid in retrospect?
    → If yes: the entry mechanics are working.
    → If no: what was the false signal? Note for the pattern library.
□ Hypothesis result: [Confirmed / Invalidated / Refined]
    → Write the refined hypothesis if the result was unexpected.
```

### STEP 3 — RESIDUE CHECK

```
□ Has the residue from Step 1 cleared?
    → If yes: the session experiment is complete. Calibration is clean.
    → If no: do NOT open a new trade. The residue is still active.
    → Perform the body check from the pre-session protocol.
    → If residue persists: session is done. Close the charts.
```

### STEP 4 — THE DISSOLUTION

The final act of every trade session:

Read this internally or aloud:

> "This trade is complete. Its outcome is data. The data has been logged.
> I release the result back into the field.
> The next session begins from the same foundation.
> The Observer remains."

## KILL CONDITIONS FOR A SESSION

Stop trading immediately and run the post-trade calibration if any of the following occur:

```
□ Two consecutive losses at the stop level → something in the read is off
□ Any single trade with a loss larger than 2% of equity → sizing protocol breached
□ Emotional decision made (entered without checklist, moved a stop, revenge-traded)
□ Three failed entries at the same level → the market is telling you the read is wrong. Listen.
□ The Observer position is lost and cannot be recovered within 5 minutes of calibration
□ Any external life event has caused acute emotional state during the session
```
