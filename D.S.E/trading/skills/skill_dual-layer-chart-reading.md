## PURPOSE

This skill operationalizes the dual-layer reading protocol from `dse_framework_orthodox-vs-esoteric-chart-reading.md`. It provides a step-by-step, timed procedure for reading any chart on any timeframe using both the orthodox (L1-3) and esoteric (L4-5) lenses simultaneously.

Time required: 15-25 minutes for a full multi-timeframe read.
Run after: `skill_observer-calibration.md` (always run observer calibration first).

## SKILL EXECUTION PROTOCOL

### PHASE 1 — ESOTERIC CONTEXT BRIEF (5 minutes)
*(Run BEFORE opening any chart for technical analysis)*

Open `dse_framework_rays-crosses-markets.md` Section 1-3 and answer:

```

   R[_] — because: [one sentence reason]

2. SUPPORTING RAYS:
   R[_] and R[_]

4. DOMINANT CROSS (current macro):
   [ ] Mutable dominant (chaos, no trade)
   [ ] Fixed dominant (trend intact, follow)
   [ ] Cardinal dominant (transition approaching, watch for entry)

5. ASTRONOMICAL CONTEXT (from dse_framework_astronomical-trading-calendar.md):

   Key upcoming event: _______________ on _______________

6. NARRATIVE PHASE (from dse_framework_macro-narrative-reading.md):
   Current dominant narrative: _______________
   Narrative phase: 1 / 2 / 3 / 4 / 5
   Field truth divergence: [ ] Aligned [ ] Diverging — explain: _______________

```


The 4H chart shows the intraday structure and the most recent Cardinal Cross decision zone.

```

Consistent with daily bias? [ ] Yes [ ] No (explain: _______)
```

#### Step 2D — 1H (2 minutes)
The 1H chart shows the current market structure within the 4H context.

```
1H STRUCTURE:
[ ] Price is near the 4H OB zone (accumulation — potential entry)
[ ] Price is in the middle of the range (no clear entry)
[ ] Price is at a premium zone (wrong location for longs / right for shorts)
[ ] Price has made a 1H BOS in the direction of daily bias (confirmation)

1H entry zone: _________ to _________

1H inducement visible: [ ] Yes — _________ [ ] No
```

#### Step 2E — 15M (2 minutes, for entry precision)
Use 15M only when a valid setup has been identified on higher timeframes.

```
15M ENTRY TRIGGER CHECK:
Is there a valid 15M confirmation signal?
[ ] Pin bar / hammer / shooting star at the OB zone
[ ] Bullish / Bearish engulfing at the entry zone
[ ] Fair Value Gap filling with rejection at midpoint
[ ] Asian range swept, London reversal forming

Exact entry level: _________
Exact stop level: _________ (below structural invalidation, not arbitrary)
Exact target level: _________ (next liquidity pool / structural high or low)
R:R ratio: _______ : 1 (minimum 2:1 required to proceed)
```

### PHASE 3 — CONVERGENCE CHECK (3 minutes)

This is the core of the dual-layer skill: testing whether orthodox and esoteric layers agree.

```
CONVERGENCE MATRIX:

ESOTERIC:
[ ] Ray energy supports this trade direction
[ ] Cycle phase supports this trade direction
[ ] Cross identification confirms (Cardinal Cross at the OB = yes / Mutable dominant = no)
[ ] Astronomical timing is clear (no station within 2 weeks, no eclipse within 6 weeks)
[ ] Narrative phase supports (Phase 1-3 = yes / Phase 4-5 = reduce size)

ORTHODOX:
[ ] Daily bias confirmed
[ ] 4H OB zone identified and price is at or approaching it
[ ] FVG above (for longs) or below (for shorts) unfilled = magnet in place
[ ] Structure is intact (HH+HL / LH+LL not broken)
[ ] Valid 15M trigger signal present

CONVERGENCE SCORE:
Esoteric boxes checked: ___ / 5
Orthodox boxes checked: ___ / 5
Total: ___ / 10

SCORE INTERPRETATION:
8-10: Maximum confidence — standard position size
6-7: Moderate confidence — half position size (0.5-1% risk)
4-5: Low confidence — observation only, no trade
< 4: Do not trade. The layers are not aligned.
```

### PHASE 4 — HYPOTHESIS STATEMENT (2 minutes)

If convergence score ≥ 6, write the integrated hypothesis:

```
INTEGRATED HYPOTHESIS:

Orthodox layer:
"[Pair] is in [HH+HL / LH+LL] on the daily chart.
The 4H OB at [level] is the Cardinal Cross decision zone.
An FVG at [level] remains unfilled above [below] — the structural magnet.
Entry trigger: [specific candle pattern] at [specific level]."

Esoteric layer:
"This is an R[_] expression during the [Polarization/Sweep/approaching Crisis] phase.
The [Mutable/Fixed/Cardinal] Cross is dominant.
The planetary context: [transit].
The narrative is in Phase [N] — [aligned with / diverging from] the field truth."

Integrated statement:
"This is an R[_] expression on [pair] during [cycle phase],
where the [OB/FVG/BOS] at [level] represents the Cardinal Cross decision zone,
supported by [L3 confirmation — COT/yield/DXY position].
Confirmed when [specific trigger].
Stop: [level] — invalidated by [structural condition].
Target: [level] for [R:R] risk-reward.
Position size: [% based on convergence score]."

HYPOTHESIS STATUS:
[ ] Complete — 6 components present
[ ] Incomplete — missing: _______________ → OBSERVATION ONLY
```

---

### PHASE 5 — WATCH OR EXECUTE

If the hypothesis is complete and the convergence score is ≥ 6:
- Set alerts at the entry level (do NOT watch the chart continuously — this is Mutable Cross behavior)
- Return to the chart when the alert fires
- Confirm the 15M trigger signal is present before entering
- Execute with the position size determined by the convergence score

If the hypothesis is complete but the entry trigger is not yet present:
- The hypothesis is valid — WAIT for the trigger
- Do not pre-enter "because the level is close"

If the hypothesis is incomplete (any component missing):
- OBSERVATION MODE — no trade
- Log what was observed: which component was missing and why
- This observation itself has value — it refines the hypothesis for the next session
