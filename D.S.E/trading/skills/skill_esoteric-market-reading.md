## PURPOSE

This skill provides the structured protocol for performing a complete esoteric market reading — identifying the active Ray, the dominant Cross, the cycle phase, and the current sacred/reactive planetary configuration. The output feeds into the dual-layer chart reading skill and the session calibration.

The **full version** (30 minutes) is run once per week — ideally Sunday as part of the weekly prep ritual.
The **abbreviated version** (5 minutes) is run daily as part of Phase 3 (Layer 4 check) in the Observer Calibration Protocol.

### MODULE 2 — CROSS IDENTIFICATION (5 minutes)

For the primary pair being tracked:

```
PAIR: _______________

MUTABLE CROSS FINGERPRINT (wicks):
- Average upper wick size vs body: Large / Normal / Small
- Average lower wick size vs body: Large / Normal / Small
- Wick direction: Upper wicks dominant (resistance above) / Lower wicks dominant (support below)
- Mutable Cross emotional state: Fearful (selling wicks) / Hopeful (buying wicks) / Confused (both)
- Mutable Cross is currently positioned: LONG / SHORT / MIXED (from retail sentiment data)

FIXED CROSS FINGERPRINT (bodies):
- Average body size: Large (conviction) / Small (uncertainty) / Mixed
- Body direction consistency: Consistent bullish / Consistent bearish / Mixed
- Fixed Cross trend: HH+HL / LH+LL / Ranging
- Is the Fixed Cross mandate supporting this direction? (COT large spec positioning): YES / NO

CARDINAL CROSS FINGERPRINT (the OB zone):
- Most recent Cardinal Cross decision (the last OB): _____________ at level _____________
- Is this zone currently being defended? (Price respecting the OB): YES / NO / NOT TESTED YET
- Where would the Cardinal Cross set the next trap? (The inducement level): _____________

DOMINANT CROSS THIS WEEK:
[ ] Mutable (high volatility, wicks, chaos) → Crisis phase, reduce size
[ ] Fixed (consistent bodies, clear trend) → Sweep phase, trend follow
[ ] Cardinal (quiet accumulation before a move) → Transition approaching, watch for entry
```
