# WF_DSC — TRIGGER & ORIENTATION
**D.S.C | Load this first. Then load only the active stage file.**

---

## WHAT THIS WORKFLOW IS

The gateway sequence for every project entering the Pandora OS. No project moves into active build without passing through all four stages. Each stage has a gate — the idea does not proceed until it is passed.

---

## TRIGGERS

- A new idea that demands attention and wants to become a project
- An external opportunity that needs evaluation
- An existing project that has lost direction and needs re-entry from Stage 1
- A request from another department to initiate something new

---

## INPUT REQUIRED BEFORE STARTING

```
IDEA/OPPORTUNITY:  State it plainly in one sentence.
SOURCE:            Internal inspiration / external opportunity / department need
URGENCY SIGNAL:    Is there a time element or is this open?
ENERGY LEVEL:      1–10 — how much pull does this have right now?
DEPARTMENT ORIGIN: Which department does this primarily belong to?
```
