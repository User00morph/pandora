# WF_DSC — STAGE 1: NIGREDO
**Raw Material Extraction — See the idea clearly before evaluating it.**

---

## WHAT THIS STAGE IS

Most ideas arrive distorted — inflated by excitement or deflated by doubt. This stage strips both distortions and looks at what is actually there. Nothing is filtered. The raw material is received exactly as it arrives.

---

## ACTION — Answer each of these without editing

**1. PLAIN STATEMENT**
Describe this idea in one sentence. Not the vision — the actual thing.
> "A YouTube channel about sovereign living" — not "a media empire that changes consciousness"

**2. ORIGIN TRACE**
Where did this come from? A problem experienced? Something consumed that sparked it? A market gap? An instruction from inner intelligence? Trace it honestly — origin reveals motivation.

**3. WHAT PROBLEM DOES IT SOLVE?**
For Morph? For others? If it solves no problem — what does it create or express? There must be a genuine answer here.

**4. RAW OBSTACLES**
Without filtering — what stands between now and this existing? List every obstacle immediately. Do not problem-solve yet. Just name them.

**5. SIMILAR THINGS THAT EXIST**
What already exists that is similar? Not a reason not to proceed — information about the landscape.

---

## OUTPUT

A Nigredo document — raw, unedited answers to all five. No polish. This is the ore, not the refined metal.

File as: `dsc_project_[name]_nigredo.md`

---

## GATE

> Does enough raw material exist to justify purification?
> If the plain statement cannot be written in one sentence — the idea is not ready. Let it develop and return.
