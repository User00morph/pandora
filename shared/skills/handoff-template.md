# Department Handoff Template
**Standard format for passing work between departments**

---

```markdown
# HANDOFF: [Source Dept] → [Receiving Dept]

## ORIGIN
- **Department:** [source department code]
- **Stage completed:** [which stage produced this]
- **File:** [path to the output file]

## PAYLOAD
- **Topic:** [what this is about]
- **Summary:** [2-3 sentence summary of what was produced]
- **Confidence level:** [ESTABLISHED / PROBABLE / POSSIBLE / INVESTIGATING]

## REQUEST
- **What the receiving department should do with this:** [specific action]
- **Urgency:** [immediate / standard / when-available]
- **Expected output format:** [what the receiving department should produce]

## CONTEXT
- **Source files for reference:** [list of files the receiving department may need]
- **Related work in progress:** [anything else in flight that connects]

## TRACE
- **Originated from:** [what triggered the original work]
- **Will feed into:** [what downstream work this enables]
```

---

*Shared template — standardized handoffs prevent information loss between departments.*
