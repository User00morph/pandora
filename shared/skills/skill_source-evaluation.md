# SKILL — SOURCE EVALUATION
**Loadable by: D.R.D, D.S.S, D.B.S, D.I.I, D.O.M**

---

## WHAT THIS SKILL IS

The ability to assess the quality, reliability, and tier of any information source before using it as a foundation for conclusions. Loaded whenever research is being conducted or claims are being evaluated.

---

## THE D.R.D SOURCE TIER SYSTEM

```
TIER 1 — Primary sources
Original research, primary texts, archaeological records,
peer-reviewed studies with no major conflicts of interest,
direct transcripts or documents

TIER 2 — Credentialed secondary sources
Academic books, peer-reviewed review articles,
credentialed experts whose work can be evaluated

TIER 3 — Serious independent researchers
Non-credentialed researchers with documented methodology,
primary source engagement, transparent reasoning,
consistent track record

TIER 4 — Secondary claims
Information that cannot be traced to a primary source,
claims with unclear origin, aggregator content

TIER 5 — Unsourced assertion
No traceable origin, no methodology, no primary source
Used only to identify what is being claimed —
never as factual foundation
```

---

## EVALUATION PROCESS

For each source encountered:

```
1. IDENTIFY the source type and assign initial tier
2. CHECK the source's relationship to the topic
   (author, funder, incentive structure)
3. TRACE the claim back — can it reach a primary source?
   If yes → tier holds or upgrades
   If no → tier drops
4. NOTE limitations explicitly
   (sample size, date, methodology, funding, bias)
5. FLAG contradictions with other sources at same or higher tier
```

---

## RULES

```
- A resonant claim from a Tier 4 source is still Tier 4
- A boring claim from a Tier 1 source is still Tier 1
- Tier is determined by methodology, not by how true it feels
- Contradicting evidence is logged at full strength, never softened
- "No evidence found" is a valid finding — do not fill gaps with speculation
```
