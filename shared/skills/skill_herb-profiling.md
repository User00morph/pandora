# SKILL — HERB PROFILING
**Loadable by: D.B.S, D.R.D**

---

## WHAT THIS SKILL IS

The structured process for researching and documenting a medicinal herb to a standard that is both practically useful and honest about evidence quality. Loaded when adding to the Sacred Herb Library or evaluating an herb for a specific application.

---

## THE PROFILE TEMPLATE

```
1. IDENTITY
   - Common name(s)
   - Latin/botanical name
   - Family
   - Part(s) used (root, leaf, flower, seed, bark, whole plant)

2. TRADITIONAL USE
   - Which cultures used this herb and for what purposes?
   - How was it traditionally prepared?
   - How far back does documented use go?
   - Source tier for traditional claims (usually Tier 2-3)

3. ACTIVE COMPOUNDS
   - Known bioactive compounds
   - Mechanism of action (where understood)
   - Documented effects on specific body systems

4. MODERN EVIDENCE
   - Clinical studies (summarize with sample size, methodology, findings)
   - In vitro / animal studies (note limitations of translation to humans)
   - Systematic reviews or meta-analyses if available
   - Source tier for each study referenced

5. PREPARATION METHODS
   - Tea / infusion / decoction
   - Tincture (alcohol or glycerin base)
   - Capsule / powder
   - Topical application
   - Dosage ranges (traditional and clinical)

6. SAFETY
   - Known contraindications
   - Drug interactions
   - Pregnancy / nursing considerations
   - Toxicity thresholds if known

7. SOURCING
   - Grow-your-own potential (climate, difficulty)
   - Quality markers when purchasing
   - Known reliable suppliers (if identified)

8. SOVEREIGN ASSESSMENT
   - Overall evidence quality (strong / moderate / traditional only / speculative)
   - Practical value for the OS (high / moderate / low / reference only)
   - Open questions that need further research
```
