## HOW TO READ THIS INDEX

| Column | Meaning |
|--------|---------|
| **Topic** | Research subject (links to active file) |
| **Domain** | One of the 6 research domains |
| **Stage** | Current pipeline position: `raw-extract` → `decoded` → `reconstructed` → `deployed` |

## PIPELINE STAGES

```
STAGE 1 — RAW EXTRACT        research/[domain]/
  Raw information gathered, sub-questions mapped, sources identified.
  File prefix: drd_research_

STAGE 2 — DECODED             deconstructions/[domain]/
  Evidence extracted, contradictions logged, mainstream claims examined.
  File prefix: drd_decode_

STAGE 3 — RECONSTRUCTED       reconstructions/[domain]/
  Sovereign position built, confidence levels assigned, framework complete.
  File prefix: drd_reconstruct_

STAGE 4 — DEPLOYED            briefs/
  Refined data packaged and delivered to destination department(s).
  File prefix: drd_brief_
```
