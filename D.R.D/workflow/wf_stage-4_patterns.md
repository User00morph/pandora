# WF_DRD — STAGE 4: PATTERN RECOGNITION + CONCEPT DISTILLATION
**Find what the evidence actually shows. Then distill it to its core.**

---

## PART A — PATTERN RECOGNITION

With all evidence extracted, step back and look at the whole picture.

**Answer these five questions:**

**1. WHERE IS THERE GENUINE CONSENSUS?**
Multiple Tier 1–2 sources pointing the same direction = high confidence finding.

**2. WHERE IS THERE GENUINE DISAGREEMENT?**
Tier 1–2 sources contradicting each other = document the disagreement honestly. Do not pick a side without examining why the disagreement exists.

**3. WHERE IS THE EVIDENCE THIN?**
Only Tier 3–4 sources available = finding is possible, not established.

**4. WHERE IS THE EVIDENCE ABSENT?**
Widely claimed but no traceable primary source = Tier 5. Flag as assertion, not finding.

**5. WHAT IS THE STRONGEST CASE AGAINST?**
Even if a conclusion seems clear — articulate the strongest version of the opposing evidence. If the opposing case cannot be articulated, the research is incomplete.

**Output:** Pattern summary organized by confidence level:
`ESTABLISHED / PROBABLE / POSSIBLE / CONTESTED / NEEDS EVIDENCE`

> Note on NEEDS EVIDENCE: not dismissed — flagged as requiring more sourcing. Include what specific evidence would strengthen or resolve the claim.

## PART B — CONCEPT DISTILLATION

**Load: `shared/skills/skill_concept-distillation.md`**

Before building the sovereign position, distill the core concepts that emerged from pattern recognition. Reconstruction is built on clearly understood foundations — not half-absorbed complexity.

For each key concept identified in Part A:

1. **EXTRACT THE ROOT** — one sentence, no jargon
2. **IDENTIFY THE PILLARS** — 3–5 load-bearing elements
3. **MAP THE RELATIONSHIPS** — how pillars connect to each other
4. **BUILD THE LAYERS:**
   - Layer 1 (The Seed): 1–3 sentences, zero background needed
   - Layer 2 (The Scaffold): ~1 page, terms defined inline
   - Layer 3 (The Full Architecture): complete with nuance and edge cases
5. **DISTORTION CHECK** — does Layer 1 still carry Layer 3's truth?

**Output:** Concept distillation(s) embedded in or appended to the decoded file in `deconstructions/[domain]/`. These distillations feed directly into Stage 5.
