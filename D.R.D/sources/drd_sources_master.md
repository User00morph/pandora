# D.R.D — MASTER SOURCE REGISTRY

**Central catalog of all sources evaluated across D.R.D research.**
**Every source used in any research file gets registered here.**

---

## HOW THIS WORKS

This is the **central registry**. Every source that enters D.R.D gets logged here once, with its tier assignment and the research topics it was used in.

Each domain also has its own source file (`drd_sources_[domain].md`) with domain-specific detail. The master registry is for cross-domain lookup — when you need to check if a source has already been evaluated, start here.

---

## SOURCE TIER DEFINITIONS

| Tier | Type | Weight | Use |
|------|------|--------|-----|
| 1 | Primary: original documents, archaeological findings, genetic studies, peer-reviewed with no major COI | Highest | Foundation-level evidence |
| 2 | Credentialed secondary: academic books, review articles, credentialed experts with evaluable work | High | Strong supporting evidence |
| 3 | Serious independent researchers: documented methodology, primary source engagement, transparent reasoning | Medium-High | Supporting evidence with notation |
| 4 | Secondary claims: untraceable to primary source, unclear origin, aggregator content | Medium | Investigation leads only |
| 5 | Unsourced assertions: no traceable origin, no methodology | Lowest | Claim mapping only — never cited as fact |
