# Source Quality Tiers
**Layer 3 Reference — Evidence standard across all departments**

---

| Tier | Source Type | Weight | Use |
|------|------------|--------|-----|
| 1 | Primary: original documents, archaeological findings, genetic studies, RCTs | Highest | Foundation for all claims |
| 2 | Peer-reviewed academic research, credentialed secondary sources | High | Strong evidence base |
| 3 | Independent scholars with documented, verifiable evidence trails | Medium-High | Serious investigation |
| 4 | Alternative researchers making specific, testable claims | Medium | Investigation only |
| 5 | Single-source, anecdotal, or unverified material | Investigate only | Never presented as fact |

## RULES

- All outputs clearly state the tier of their primary evidence
- The confidence level of every conclusion is stated honestly
- Tier 5 material is captured for investigation, never presented as established fact
- Contradicting evidence is logged at full strength — never softened
- Source tier is determined by methodology and accountability, not by whether the finding is significant

## CONFIDENCE LEVELS

| Level | Meaning |
|-------|---------|
| ESTABLISHED | Multiple Tier 1-2 sources across independent frameworks agree |
| PROBABLE | Strong evidence but not definitive |
| POSSIBLE | Evidence exists but is thin or contested |
| CONTESTED | Tier 1-2 sources directly contradict each other |
| UNSUPPORTED | No traceable primary evidence |
