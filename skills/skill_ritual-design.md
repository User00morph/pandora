# SKILL — RITUAL DESIGN
**Loadable by: D.O.M, D.P.S.A**

---

## WHAT THIS SKILL IS

The ability to design a structured ritual or practice sequence for a specific intention. A ritual here is not performance — it is a deliberate sequence of actions designed to focus consciousness, direct energy, and produce a specific internal or external shift. Loaded when designing new practices or formalizing existing ones.

---

## THE DESIGN PROCESS

```
1. INTENTION STATEMENT
   What is this ritual designed to accomplish?
   State it clearly and specifically.
   "Align with the new moon energy" is vague.
   "Set three specific intentions for the lunar cycle
    and anchor them in written and symbolic form" is specific.

2. STATE REQUIREMENTS
   What internal state does the practitioner need to be in
   for this ritual to work?
   - Calm, focused, energized, receptive?
   - How is that state induced? (breathwork, meditation, movement)
   - How long does state induction take?

3. COMPONENTS
   What elements does this ritual use?
   - Physical tools (candles, sigils, crystals, herbs, water)
   - Written elements (statements, lists, letters)
   - Spoken elements (invocations, affirmations, mantras)
   - Movement (specific gestures, positions, dance)
   - Timing (moon phase, time of day, planetary hour)

4. SEQUENCE
   The exact order of operations:
   - Opening (how it begins — state induction, space preparation)
   - Body (the core actions — what is done, said, written, visualized)
   - Closing (how it ends — grounding, sealing, gratitude, cleanup)

5. DURATION
   How long does this ritual take?
   Be realistic — a ritual that is too long will not be repeated.

6. FREQUENCY
   How often is this performed?
   One-time / daily / weekly / lunar cycle / seasonal / as needed

7. DOCUMENTATION
   How will results be tracked?
   What does the practitioner observe after performing this ritual?
   Where is that observation logged?
```

---

## RULES

```
- Every ritual must have a specific, stated intention
- The practitioner designs their own system — no inherited authority required
- If a ritual stops producing results, redesign it — do not repeat dead forms
- Simpler is usually more powerful — do not add complexity for its own sake
- Direct experience is the authority, not tradition or aesthetics
```
