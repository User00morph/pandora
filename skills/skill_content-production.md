# SKILL — CONTENT PRODUCTION PIPELINE
**Loadable by: D.C.E, D.S.E**

---

## WHAT THIS SKILL IS

The end-to-end process for taking a content idea from raw concept to published output across any medium — video, written, audio, visual, social. Loaded when producing content that will be published or distributed.

---

## THE PIPELINE

```
STAGE 1 — BRIEF
What is this piece?
- Medium (video / article / social post / podcast / course)
- Core message (one sentence)
- Target audience
- Intended outcome (what should the audience think, feel, or do?)
- Length / format constraints

STAGE 2 — RESEARCH
Does this piece require sourced information?
If yes → route to D.R.D workflow or load skill_source-evaluation.md
If no → proceed with personal knowledge and experience

STAGE 3 — SCRIPT / DRAFT
Produce the first complete version.
- For video: full script or structured outline with key points
- For written: complete first draft
- For social: complete post(s)
- For audio: episode outline with talking points
Use skill_copywriting.md at this stage.

STAGE 4 — REVIEW
Read/watch the draft with fresh eyes.
- Does it deliver the core message from the brief?
- Is anything included that does not serve the core message?
- Does the voice match Morph's authentic expression?
- Is the structure clear — can someone follow it?

STAGE 5 — PRODUCTION
Produce the final version.
- Video: record, edit, thumbnail, metadata
- Written: final edit, formatting, publish
- Social: final copy, visuals, scheduling
- Audio: record, edit, show notes

STAGE 6 — PUBLISH AND FILE
Publish to the appropriate platform.
File the final version:
  dce_[medium]_[title]_final.md
File any supporting materials alongside it.
```

---

## RULES

```
- Never publish from Stage 3 — the draft is not the final
- Never skip the brief — a piece without a clear brief drifts
- The review stage is where 80% of quality improvement happens
- File everything — a published piece with no filed source is lost knowledge
```
